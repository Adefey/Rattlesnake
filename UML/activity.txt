1. На сервер приходит запрос в json, содержащий vector<Block> и vector<Parameter>
2. Запрос парсится и по адресу отправителя, вектору блоков и параметров создается экземпляр User. В экземпляре класса NetUtils содержится адрес юзера
3. Экземпляр User отправляется в waiting_users
4. Если в executing_users освобождается место, экземпляр User отправляется туда и юзером управляет process_user
5. На юзера создается машина (методом MachineFactory)
6. Создается 'экземпляр Machine и указатель на него пишется в machines_online
7. Экземпляр Machine через метод process_make_job запускает все блоки
8. По окончанию очередного блока, данные пишутся в User.vars и запускается новый процесс.
9.  в process_user оказывается юзер с полностью заполненным поем vars
10. Готовый экземпляр User переходит в ready_users
11. Когда до него доходит очередь, с помощью парсера и NetUtils составляется json по vector<User.vars> и отправляется
12. Экземпляр User уничтожается
